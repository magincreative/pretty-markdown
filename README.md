# pretty-markdown

## TLDR:
This is a light weight in-browser tool to "prettify" markdown files created by AI tools like Claude and ChatGPT.

## What is it?
Who hates having to read markdown files in IDEs where the colour contrast is glaring and words blur together. Great UI for writing code but not so much when reading longer form sentences and paragraphs. Enter Pretty-Markdown.

A simple in-browser tool where you locally upload a markdown file and have it rendered back to you in CSS styled HTML.

## How to use?
1. Download the `pretty-markdown.html` file to your computer.
2. Open it in your browser
3. Upload your markdown file
4. Watch as it is rendered in easy to read styling.
